## 📊 Expected Output Format

Each entry in the JSONL file should now contain:
```json
{
  "input": "system\nYou are...\nuser\nQuestion: Who wrote...",
  "output": "assistant\n<think>Let me search...</think><tool_call>...",
  "gts": {"target": ["Sam-I-am"]},
  "score": 1.0,
  "step": 1,
  "assistant_turns": 3,
  "user_turns": 3,
  "pred_ans": "Sam-I-am"
}
```

**Field Descriptions**:
- `assistant_turns`: Number of times the model generated responses (includes tool calls)
- `user_turns`: Number of times the system provided input (includes tool responses)
- `pred_ans`: Extracted answer from `<answer>` tags, or `None` if not found
