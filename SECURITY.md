# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in LiteResearcher, please report it
**privately**. Do not open a public issue for security problems.

- Use GitHub's [private vulnerability reporting](https://github.com/simplexai-labs/LiteResearcher/security/advisories/new)
  for this repository, or
- Email the maintainers at **security@simplex-ai.com** with a description of the
  issue and steps to reproduce.

Please include:

- The affected component (`inference` / `training` / `datagen` / `environment`).
- A description of the vulnerability and its potential impact.
- Steps to reproduce, and a proof-of-concept if available.

We will acknowledge your report within a reasonable timeframe and keep you
updated on remediation progress. We ask that you give us a reasonable window to
address the issue before any public disclosure.

## Scope

This policy covers the code in this repository. Released model weights and
datasets hosted on Hugging Face are governed by their respective model/dataset
cards.
